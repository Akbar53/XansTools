## Getting Started

Membutuhkan Python 1 / Python 3

### Cara Install
Clone git ini

```bash
pkg install update&&upgrade
pkg install python
pkg install python3
pip install --upgrade pip
pip install requests
pip install bs4
pip install urllib3
git clone https://github.com/natxanss/XansTools.git
cd XansTools
python xans-tools.py
```
